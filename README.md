# Steel Boss: Factory Tycoon

Expo + React Native + TypeScript idle tycoon game where players build a steel factory empire.

## Features
- Factory, Machines, Orders, Lean Upgrades, Shop, and Settings screens
- Manual production and automated income per second
- Machine upgrade progression and scaling costs
- Contracts/orders with cash rewards
- Lean upgrades (5S, TPM, Kaizen, Kanban, Foreman)
- AsyncStorage local persistence
- Offline earnings based on away time
- Bottom tab navigation and dark industrial UI
- Placeholder rewarded ads and in-app purchases

## Run (Android)
```bash
npm install
npm run android
```

## Development
```bash
npm run start
npm run typecheck
```
